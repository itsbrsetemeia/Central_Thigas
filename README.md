# 🌌 Thigas Central

![Thigas Central](https://img.shields.io/badge/Status-Em%20Desenvolvimento-blue)

Um site pessoal criado para apresentar meus projetos, universos criativos, redes sociais e minha identidade como criador, escritor e desenvolvedor.

---

## 📌 Sobre o projeto

**Thigas Central** é um portfólio pessoal desenvolvido para reunir:

- 🚀 Projetos de desenvolvimento
- 🌌 Universo de histórias e criação (Vertigma)
- ✍️ Escrita criativa
- 🌐 Redes sociais
- 🎧 Playlist e identidade sonora

O objetivo é transformar ideias em uma experiência visual moderna e interativa.

---

## ✨ Funcionalidades

✅ Design moderno com efeito glassmorphism  
✅ Layout responsivo para celular e computador  
✅ Animações suaves ao rolar a página  
✅ Seções organizadas por categoria  
✅ Integração com Font Awesome  
✅ Player do Spotify integrado  
✅ Links para redes sociais
